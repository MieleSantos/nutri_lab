# nutri_lab

Projeto Django da PyStack Week 4.0
Que vai aborda os topicos:

- Autenticação de usuário
- Criação de Models
- Conexão com banco de dados
- Listagem de Dados
- Filtros
- Upload de imagens



Projeto de um sistemas para nutricionista, em vai ter as opções:

- Cadastrar nutricionista
- Logar no sistemas
- Cadastrar paciente
- Gerenciar informações sobre paciente
    - peso
    - altura
    - % de musculo
    - % de gordura
- Gerenciar dados laboratoriais
    - Colesterol HDL
    - Colesterol LDL
    - Colesterol total
    - Triglicerídios

- Grafico de acompanhamento
- Plano Alimentar 
# Tecnologias usadas

- Poetry
- Django
- Docker
- Postgres
